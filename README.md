# jannes_site
Test website for Jannes

## Directory Structure

This website includes the following directories:

- **css/** - Stylesheets for the website
- **js/** - JavaScript files for interactivity
- **images/** - Image assets (photos, icons, etc.)
- **fonts/** - Custom font files
- **assets/** - General assets and resources
- **videos/** - Video content

## Getting Started

Simply open `index.html` in a web browser to view the website.
