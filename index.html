How a website works

1. HTML is the structure

It is like the bones of a house: headings, paragraphs, buttons, images.

2. CSS is the style

It is paint, fonts, spacing, and all the pretty details.

3. JavaScript is the sparkle

It makes buttons do things and adds tiny surprises.

Publish it step by step

  1. Make an account on GitHub.
  2. Create a repo with a name on your account.
  3. Create an account on Netlify.
  4. Choose a free domain or buy a domain.
  5. Follow the instructions on Netlify to link your GitHub repo and publish the site.
  6. Install VS Code, then install Git and log in to your GitHub account.

    Windows (PowerShell):

    winget install --id Git.Git -e

    Mac (Terminal):

    brew install git

    Login to GitHub in Git:

    git config --global user.name "Your Name" git config --global user.email "you@example.com" gh auth login
  7. Press Shift + Ctrl + P (or Cmd + Shift + P on Mac) and type "clone".
  8. Choose the website repo to clone.
  9. Start an AI session from Jeedee23's team.
  10. Choose GPT 5.2 Codex and select "agent mode".
  11. Ask the AI to create your website, but first make a text document with everything you want on the site. Adjust it as you go.
